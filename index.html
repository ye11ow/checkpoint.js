Checkpoint.js is a Javascript library that allows you to create a user interface like the above one in seconds.


Here are some highlights:

  • Pure Javascript/Css, no other library/framework needed
  • 4KB Javascript, 2KB CSS
  • Customize your own CSS stylesheets (Using LESS)
  • Open source under MIT license

Checkpoint.js can be used in the following scenarios:

  • Status Indicator
  • Progress Bar
  • Timeline

1. Init

checkpointJs(selector, [options]);

Assign a DOM selector (container) to checkpoint.js. Notice that currently, the container has to be 80px in height. 

var checkpoint = checkpointJs("#checkpoint");

2. Set data

setStages([stage, stage, stage...]);

For each stage, there are four attributes:

  • id [string]: the id will be set to DOM
  • title [string]: the title display on this stage
  • description [string]:
  • onStageCallback [function]: the function that will be called after entering this stage

setStage(index, stage);

For each stage, there are four attributes:

  • id [string]: the id will be set to DOM
  • title [string]: the title display on this stage
  • description [string]:
  • onStageCallback [function]: the function that will be called after entering this stage


checkpoint.setStages([{
    id: "introduction",
    title: "What is?",
    description: "What is Checkpoint.js?",
    onStageCallback: changeTab
},{
    id: "examples",
    title: "Examples",
    description: "Some Examples",
    onStageCallback: changeTab
}, {
    id: "usage",
    title: "Usage",
    description: "Usage",
    onStageCallback: changeTab
}]);

3. Draw

init(defaultStage);

Draw checkpoint.js on the UI. Notice that currently, the container has to be visible while drawing checkpoint.js, otherwise checkpoint.js cannot detect the width of container. 

checkpoint.init(0);

4. Control

atStage(stageIndex);

Set checkpoint.js to a specific stage. The index starts from 0.

next();

Go to next stage.

prev(stageIndex);

Go back to previous stage. 


// by default, the checkpoint is at stage 0.
checkpoint.atStage(2); // now at stage 2
checkpoint.next(); // now at stage 3
checkpoint.prev(); // back to stage 2 again
